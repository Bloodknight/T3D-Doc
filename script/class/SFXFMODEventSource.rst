SFXFMODEventSource
==================

A sound source controller playing an FMOD Designer event (SFXFMODEvent).

Inherit:
	:doc:`SFXSource`

Description
-----------

A sound source controller playing an FMOD Designer event (SFXFMODEvent).

FMOD event sources are internally created by the sound system to play events from imported FMOD Designer projects.
